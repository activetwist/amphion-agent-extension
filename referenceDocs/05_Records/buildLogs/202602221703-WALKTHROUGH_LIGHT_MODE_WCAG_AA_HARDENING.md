# Walkthrough: Light Mode WCAG AA Contrast Hardening

## Light Mode Verification Steps
1. Open Command Deck in browser.
2. Switch to light mode using the header theme toggle.
3. Confirm milestone badges are clearly legible in milestone and card contexts.
4. Confirm issue badges are legible at normal zoom without faded/low-opacity appearance.
5. Confirm `New Board` and danger buttons have clear, readable label contrast.

## Dark Mode Regression Check
1. Toggle back to dark mode.
2. Confirm badge/button contrast remains visually correct.
3. Confirm no layout or interaction changes were introduced.

## Targeted WCAG Result
- Targeted failing selectors now meet WCAG AA normal-text contrast thresholds.
